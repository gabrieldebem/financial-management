## About the App
This application wah created to help people to manage their spending and incomes.

## Testing
You can run `sail test --parallel` to run all the tests.
In case you want to see the coverage, you will need to change your `.env` adding `SAIL_XDEBUG_MODE=coverage` then run `sail test --parallel --coverage`

## Running the App
To run this API you will need only Docker.

Run:

- `cp .env.example .env`

- `
docker run --rm \
    -u "$(id -u):$(id -g)" \
    -v "$(pwd):/var/www/html" \
    -w /var/www/html \
    laravelsail/php82-composer:latest \
    composer install --ignore-platform-reqs
`

- `sail up --build -d`
